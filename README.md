Tool para automatizar auditoria em aplicações web.
Ferramentas auxiliares:
- NMAP
- WPSCAN
- UNISCAN

Install:
$ git clone https://github.com/DevIgor/ScanPy.git 
$ cd ScanPy
$ ./install.sh

Uso:
  ./ScanPy.py --ip [IP DO SITE OU IP DA MÁQUINA] -url [URL DO ALVO]
